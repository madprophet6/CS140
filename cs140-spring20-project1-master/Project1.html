Project 1 - Designing Traffic Island

Project Overview

This project will give you experience on the use of:

  1. Java primitives int and float
  2. Arithmetic operations
  3. Using Scanner to get user input
  4. Using the println statement

The goal of this project is to gain experience with mathematical manipulation of numeric values. You will calculate areas and volumes to determine materials required for a traffic island project. The basic design consists of prompting the user for information, receiving information, processing that information, and then displaying the result. This project will provide you with the opportunity to put together nearly all of the material you have learned so far to create a single program. Most of what you are asked to do is very similar to what you did in the Lab 1A, Lab 1B, Lab 2, and the zyBook, so look back at those labs for ideas. You will be writing code for one method — main. Refer to Lab 1A, Lab 1B and Lab 2 for sample programs with only the main method.

Background

The redesign of Kelley Square in Worcester calls for a traffic island that is being described as a “peanut” shape. You are going to write a program to design a simplified version of this traffic island.

Programs are good at performing routine mathematical calculations. By way of illustration, you will write a program to calculate the dimensions and materials needed for the new Kelley Square traffic peanut according to the design given here. In this design, the green areas represent flowerbeds and the dark gray areas are sidewalks.

Project Description and Specification

Your program should prompt the user for the following information:

  1. The measurement x (in feet).
  2. The measurement y (in feet)
  3. The recommended spacing (in feet) between plants.
  4. The depth (in feet) of the flowerbeds.

Calculate the overall length and width of the traffic island, as well as the sidewalk area and sidewalk width.

Next calculate the number of plants and the amount of flowerbed soil needed.

Finally, it should report: 1. The specifications of the traffic island (overall length and width, sidewalk area, and sidewalk width.) 2. Number of plants needed for each circular flower bed, and for the middle are between the circles. 3. Cubic yards of soil for each type of flowerbed and total cubic yards of soil for the garden.

Project Notes

To clarify the problem specifications, I provided two example runs of the program:

The formula for the area A of a circle is A = π r2, where r denotes the radius. For this calculation, you should use the value of π provided in the Java’s Math package. Simply use Math.PI. Math.PI gives you (a close approximation to) the value of π. To estimate the number of plants for a flowerbed, divide the area of the flowerbed by the area needed per plant (the square of the recommended distance between plants) and then truncate this result. To truncate a float, convert it to an int using casting which implicitly truncates the float. Be careful, the number of plants in a semicircle may not be exactly half of the plants in a full circle.

Assume that the user only inputs numbers, and that measurement y is less than measurement x, that is, you don’t need to do error checking for bad input.

Remember to convert fill to cubic yards. Here is an outline of your program:

  1. Prompt for input (use Scanner class’s nextFloat method)
  2. Do some mathematical calculations using the input numbers to get results.
  3. Print the results (use System.out.println)

Getting Started

  1. Solve the problem using paper, pencil and calculator so you understand the problem before trying to program it.
  2. Using BlueJ, create a new Project.
  3. Using the example from class notes and labs write the code to prompt the user, input in the required values, and echo (print) the values that were input.
  4. Run the program and track down any errors.
  5. Now you enter a cycle of edit-run to incrementally develop your program.
  6. Submit your final version.
